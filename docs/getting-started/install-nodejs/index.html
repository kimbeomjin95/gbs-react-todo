Node.js 설치하기

React 개발을 시작하기 위해서는 먼저 Node.js를 설치해야 합니다.

🤔 Node.js란?​

Node.js는 JavaScript를 브라우저 밖에서 실행할 수 있게 해주는 런타임 환경입니다.

React 개발에서 Node.js가 필요한 이유:

  • npm/pnpm: 패키지 관리 도구 사용
  • 개발 서버: 로컬 개발 서버 실행
  • 빌드 도구: Vite, Webpack 등의 빌드 도구 사용

📋 설치 전 확인사항​

Node.js가 이미 설치되어 있는지 확인:

node --version
npm --version

버전이 출력되면 이미 설치되어 있습니다. Node.js 20.0 이상을 권장합니다.

💻 설치 방법​

Windows​

방법 1: 공식 설치 프로그램 (권장)​

  1. Node.js 공식 웹사이트 접속
  2. LTS(Long Term Support) 버전 다운로드
  3. 다운로드한 .msi 파일 실행
  4. 설치 마법사 따라 진행
    • "Automatically install necessary tools" 체크 권장

방법 2: winget 사용​

winget install OpenJS.NodeJS.LTS

macOS​

방법 1: 공식 설치 프로그램​

  1. Node.js 공식 웹사이트 접속
  2. LTS 버전 다운로드
  3. 다운로드한 .pkg 파일 실행
  4. 설치 마법사 따라 진행

방법 2: Homebrew 사용 (권장)​

# Homebrew가 없다면 먼저 설치
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Node.js 설치
brew install node

Linux (Ubuntu/Debian)​

# NodeSource 저장소 추가
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -

# Node.js 설치
sudo apt-get install -y nodejs

Linux (기타 배포판)​

Node.js 공식 설치 가이드 참고

✅ 설치 확인​

터미널(명령 프롬프트)에서 다음 명령어 실행:

node --version
# v20.x.x 또는 그 이상

npm --version
# 10.x.x 또는 그 이상

🚀 pnpm 설치 (선택사항)​

pnpm은 npm보다 빠르고 디스크 공간을 절약하는 패키지 관리자입니다.

# npm을 사용하여 pnpm 설치
npm install -g pnpm

# 설치 확인
pnpm --version

🎯 다음 단계​

Node.js 설치가 완료되었다면 프로젝트 생성하기로 이동하세요!

❓ 문제 해결​

권한 오류 (macOS/Linux)​

# npm 전역 설치 시 sudo 없이 사용하도록 설정
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'

# .bashrc 또는 .zshrc에 추가
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
source ~/.bashrc

버전 관리 (nvm 사용)​

여러 버전의 Node.js를 사용해야 한다면 nvm 사용을 권장합니다:

macOS/Linux:

# nvm 설치
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Node.js LTS 설치
nvm install --lts
nvm use --lts

Windows: nvm-windows 다운로드 및 설치
